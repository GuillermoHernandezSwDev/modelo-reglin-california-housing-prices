# Modelo de Predicción de Precios de Vivienda en California

## Descripción

Este proyecto implementa un modelo de regresión lineal simple para intentar predecir el valor medio de las viviendas en distritos de California. Se utiliza el conjunto de datos del censo de 1990 para analizar cómo los factores geográficos y demográficos influyen en los costos de las propiedades.

## Especificaciones Técnicas

* Fuente de datos: Censo de California de 1990.
* Tipo de modelo: Regresión Lineal.
* Métricas de evaluación: R-cuadrado y Error Cuadrático Medio.

## Características

* Exploración de datos de vivienda mediante análisis estadístico.
* Limpieza de datos y manejo de valores faltantes.
* Ingeniería de características y normalización de variables numéricas.
* Análisis de correlación para identificar predictores significativos.

## Tecnologías Utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Estructura del Proyecto

1. Carga de datos: Importación del conjunto de datos de vivienda.
2. Preprocesamiento de datos: Limpieza y preparación de variables para el modelo.
3. Análisis exploratorio de datos: Visualización de distribuciones y correlaciones.
4. Implementación del modelo: Entrenamiento de un algoritmo de regresión lineal.
5. Evaluación: Valoración del rendimiento del modelo con datos de prueba.
